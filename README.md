Desenvolvido em Java

Objetivo 
  Simular o funcionamento de um micro-ondas digital permitindo:
  - Início manual (tempo e potência)
  - Programas pré-definidos
  - Validações de regra de negócio:
    Regras atendidas
     ✔️5 programas fixos
     ✔️Imutáveis
     ✔️Não podem ser alterados
     ✔️String diferente de "."
     ✔️Bloqueio de acréscimo
     ✔️Preenchimento automático de tempo e potência
     ✔️Pausa e cancelamento permitidos
  - Tratamento global de erros
  - Teste unitários das camadas
  - Estrutura arquitetural organizada

Arquitetura do Nível 2
  Projeto organizado em camadas:
  com.microondas
  - domain
  - application
  - presentation
  - infrastructure

Linguagens e ferramentas utilizadas
  - JavaFX 
  - Maven  
  - Java 11+
  - IDE Vs Code
